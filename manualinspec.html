Manual de inspecciones (Tablet)

Este manual describe el proceso recomendado para realizar inspecciones desde tablet. Está orientado a usuarios con rol de inspector (y perfiles autorizados).

Antes de empezar

1) Internet y sesión
Asegúrate de tener conexión a Internet al menos para iniciar sesión. Si vas a trabajar en campo, inicia sesión antes de salir.
2) Permisos del navegador
Permite:
Cámara (para evidencia y observaciones).
Ubicación (si aplica para registro).
3) Recomendación de uso
Usa el navegador Chrome (o equivalente) y mantén la pestaña abierta durante la captura. Evita cambiar entre muchas apps mientras guardas.

Flujo de inspección (paso a paso)

Paso 1: Abrir “Nueva inspección”
En el menú ve a:
Inspecciones → Nueva inspección
Paso 2: Elegir tipo de inspección
Selecciona uno:
Pre-trabajo
Post-trabajo
Recepción
Paso 3: Seleccionar equipo
En el campo Equipo escribe el código (ej. PUP, XO, etc.) o parte de la descripción. En tablet se muestra un listado para seleccionar.
Paso 4: Revisar el “Detalle del equipo”
Verifica:
Equipo / activo
Producto y Descripción
Serial
Si el equipo no coincide, regresa y selecciona el correcto.
Paso 5: Inspeccionar parámetros
En la tabla de parámetros:
BUENO / MALO: marca el estado.
Tipo de daño: aparece cuando marcas MALO. Selecciona el daño y, si aplica, describe en “OTRO”.
Evidencia: toma o sube foto cuando sea necesario.
Lados A/B y TEES
Algunos equipos requieren revisar más de un lado.
A/B: verás parámetros duplicados como “Área de sellado A/B” y “Espárragos y tuercas A/B”.
TEES: pueden mostrar 3 lados según el tipo (por ejemplo un patrón tipo HMH/HHM/MMH).
Revisa y captura cada lado por separado.
Paso 6: Observaciones
Al final puedes:
Escribir observaciones generales.
Tomar/Subir una foto de observaciones si aplica.
Paso 7: Guardar inspección
Presiona Guardar inspección. Espera a que termine. Si hay problemas de conexión, el sistema puede dejar la inspección pendiente para sincronizar.

Problemas comunes

No aparece el equipo
Verifica ortografía del código, intenta buscar por una parte del código o descripción. Si el equipo no está cargado en inventario, reporta a supervisión.
No se puede tomar foto
Revisa permisos de cámara en el navegador. Si hay más de una cámara, intenta “Cambiar cámara” cuando aparezca.
No guardó / quedó pendiente
Mantén la sesión abierta, revisa conexión, y vuelve a intentar. Si la inspección quedó pendiente, se intentará sincronizar cuando haya Internet.

# INVIVAPP — Checklist de capturas (expediente)

## Capturas recomendadas

- Login (pantalla inicial)
- Inicio / Dashboard (indicadores principales)
- Inventario (vista/listado)
- Detalle de un activo/equipo
- Registro de inspección (formulario)
- Listado de inspecciones
- Registro de prueba/certificado
- Listado de pruebas (con vigencias/alertas)
- Visor de evidencia (si aplica)
- Exportación de reporte (si aplica)
- Vista de roles/permisos (si hay UI visible o comportamiento restringido)

## Reglas

- Evitar mostrar:
  - correos/usuarios sensibles
  - IDs internos innecesarios
  - datos personales
  - rutas o llaves

Formato sugerido: PNG o JPG.

Aviso de Privacidad y Términos de Uso

Este documento describe cómo se tratan los datos personales y la información operativa dentro de PCT (Sistema interno de operación), así como las reglas mínimas de uso para sus usuarios.

Última actualización
10/04/2026

1) Identidad y contacto del responsable

Proveedora y Comercializadora Tabasqueña S.A. de C.V. (PCT) es responsable del tratamiento de los datos personales y de la información capturada en PCT (Sistema interno de operación).

Contacto
Correo: logistica@pc-t.com.mx
Asunto sugerido: “Sistema PCT – Privacidad / ARCO”

2) Sistema y usuarios

PCT (Sistema interno de operación) es un sistema interno para registrar y consultar información operativa relacionada con equipos, pruebas, actividades, inspecciones y trazabilidades. El acceso se controla mediante autenticación y roles (por ejemplo: admin, director, supervisor, inspector, capturista y visor).

3) Datos que se pueden tratar

Datos de cuenta y control de acceso
Correo electrónico corporativo, rol/perfil asignado, identificadores técnicos de autenticación y sesiones.
Datos operativos capturados en el sistema
Registros de actividades, pruebas, inspecciones, trazabilidades, folios, fechas, equipos, ubicaciones operativas, comentarios y evidencias asociadas (por ejemplo, fotografías o documentos cuando aplique).
Bitácoras y auditoría
Registros técnicos necesarios para seguridad, trazabilidad y soporte (por ejemplo: eventos de acceso, acciones relevantes en el sistema y metadatos técnicos mínimos).

4) Finalidades del tratamiento

Finalidades principales
- Autenticación, control de acceso y administración de roles.
- Registro, consulta y seguimiento de equipos, pruebas, inspecciones y actividades.
- Trazabilidad y evidencia de eventos operativos.
- Generación de reportes y exportaciones (por ejemplo, PDF) para operación interna.
- Envío de alertas (por ejemplo, correos de recordatorio de vencimientos o rangos de prueba) a destinatarios autorizados.

5) Bases de almacenamiento y servicios de terceros

Para operar, PCT (Sistema interno de operación) utiliza servicios de terceros estrictamente como encargados del tratamiento, con el fin de proporcionar infraestructura, autenticación, almacenamiento, envío de correos y operación del sistema.

Proveedores utilizados por el sistema
- Google Firebase (Autenticación y base de datos).
- IONOS (servicio SMTP para envío de correos).
- GitHub Pages (publicación del sitio web).
- Fly.io (hospedaje/infraestructura del backend para ejecución de alertas).

6) Transferencias

No se realizan transferencias de datos personales para fines distintos a los descritos en este aviso, salvo aquellas necesarias para la operación del sistema mediante proveedores tecnológicos (encargados) o por mandato de autoridad competente.

7) Conservación

La información se conserva durante el tiempo necesario para cumplir las finalidades operativas, de seguridad, auditoría y cumplimiento aplicable. Posteriormente podrá ser bloqueada, anonimizada o eliminada conforme a políticas internas.

8) Derechos ARCO y medios para ejercerlos

Puedes ejercer los derechos de Acceso, Rectificación, Cancelación u Oposición (ARCO), así como revocar el consentimiento cuando aplique, mediante solicitud al correo logistica@pc-t.com.mx. Para atender la solicitud, podremos pedir información para acreditar identidad y para localizar el registro correspondiente.

9) Seguridad

Se aplican medidas razonables de seguridad administrativas, técnicas y físicas para proteger la información contra daño, pérdida, alteración, destrucción o uso, acceso o tratamiento no autorizado. Ningún sistema es infalible; si identificas un incidente, repórtalo al contacto indicado.

10) Cambios al aviso

Este aviso puede actualizarse por mejoras del sistema, cambios normativos o ajustes operativos. La versión vigente estará disponible en esta misma URL.

Términos de Uso (PCT - Sistema interno de operación)

1) Acceso y roles

El acceso es exclusivo para personal autorizado. Cada usuario opera bajo un rol (por ejemplo: admin, director, supervisor, inspector, capturista, visor) y solo puede ver/editar lo permitido por ese rol.

2) Confidencialidad y credenciales

Las credenciales son personales e intransferibles. No compartas tu contraseña ni dejes sesiones abiertas. Cualquier actividad registrada bajo tu cuenta se considerará realizada por ti.

3) Uso permitido

- Capturar información real, verificable y conforme a los procesos internos.
- Respetar los formatos, fechas y folios para mantener trazabilidad.
- Reportar errores o inconsistencias para corrección.

4) Uso prohibido

- Intentar acceder a información no autorizada o evadir controles de seguridad.
- Alterar evidencia o registros con fines ajenos a la operación o sin autorización.
- Introducir contenido malicioso (software, scripts, archivos peligrosos).
- Usar el sistema para fines personales, comerciales externos o ilícitos.

5) Disponibilidad

El sistema se ofrece para apoyar la operación. Puede haber mantenimientos, mejoras o interrupciones técnicas. Se harán esfuerzos razonables para preservar disponibilidad y continuidad.

6) Alertas y notificaciones por correo

Las alertas por correo se envían a destinatarios autorizados como apoyo de recordatorios y seguimiento. La recepción puede depender de infraestructura de terceros (proveedor SMTP, filtros anti-spam, etc.). Las alertas no sustituyen la revisión operativa del sistema.

7) Cambios

PCT puede modificar funciones, roles, reglas y estos términos para mejorar el sistema o atender necesidades operativas. Al continuar usando PCT (Sistema interno de operación) después de una actualización, aceptas la versión vigente.
